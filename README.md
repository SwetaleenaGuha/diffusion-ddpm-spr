# DDPM Image Generation (Midterm Project)

## Description
This project implements a Denoising Diffusion Probabilistic Model (DDPM) for image generation.

## Features
- U-Net based diffusion model
- Training pipeline
- Image generation from noise

## Dataset
ImageNet100 (subset used due to CPU limitations)

## How to Run
python train.py

## Results
The model generates noisy outputs after limited training, which is expected for early-stage diffusion models.
